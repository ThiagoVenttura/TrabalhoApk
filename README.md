# TrabalhoApk

Projeto desenvolvido como atividade prática para aplicar conceitos básicos de desenvolvimento web e introdução a Progressive Web Apps (PWA).

A aplicação roda diretamente no navegador e pode ser instalada no celular ou computador, funcionando de forma semelhante a um aplicativo nativo. O projeto utiliza arquivos de áudio e recursos simples de interface, com foco no aprendizado.

## Objetivo

Praticar:
- Estruturação de páginas com HTML
- Interatividade básica com JavaScript
- Conceitos iniciais de PWA
- Organização de arquivos em um projeto real
- Uso de recursos multimídia na web

## Tecnologias

- HTML  
- JavaScript  
- Manifest.json  
- Service Worker  
- Arquivos de áudio (MP3) e imagem  

## Funcionalidades

- Execução direta no navegador
- Instalação como aplicativo (PWA)
- Reprodução de músicas
- Funcionamento offline básico
- Interface simples

Clonar o repositório:

Abrir o arquivo `index.html` no navegador.
